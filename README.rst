Overview
########

CSV file converter to Laravel Seeders.

|Terminal|

Installation / Usage
####################

1. To install use pip

.. code-block:: bash

    pip install csv2seed

Example
#######

.. code-block:: bash

    csv2seed --model User /home/erick/Escritorio/data-users.csv
